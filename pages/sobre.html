Sobre o projeto

Lumen AI para Soul UP

Uma experiência digital mais simples, clara e motivadora para aproximar usuários da gamificação sustentável.

?

O problema

Quando a plataforma não explica bem missões, pontos e recompensas, o usuário pode perder interesse antes de perceber valor na experiência.

✦

Nossa solução

A Lumen AI guia a jornada com linguagem simples, feedback visual e recomendações claras para o próximo passo do usuário.

↗

Como funciona

O usuário acompanha evolução, missões, pontos, níveis e recompensas em uma interface objetiva, visual e motivadora.

✓

Impacto esperado

A proposta reduz dúvidas, aumenta retenção e torna a experiência sustentável mais acessível para diferentes públicos.

Jornada simplificada

1 Acessa

O usuário entende rapidamente o objetivo da plataforma.

2 Realiza

Completa ações, tarefas e missões com orientação visual.

3 Evolui

Recebe pontos, níveis, badges e recompensas.

4 Retorna

Percebe valor e continua interagindo com a Soul UP.

Tecnologias utilizadas

O projeto foi desenvolvido com tecnologias fundamentais de Front-End, mantendo uma estrutura simples, organizada e adequada para uma primeira versão funcional da solução.

HTML5 CSS3 JavaScript GitHub Design Responsivo
